# Exercise 04: Copilot for Sales and Service

## Exercise learning objectives
- Enable and configure Copilot features in Dynamics 365 Sales.
- Enable and configure Copilot features in Dynamics 365 Customer Service.
- Use Copilot to summarize records, draft emails, and suggest next-best actions.
- Validate AI-assisted interactions in both Sales and Service contexts.

## Licensing and environment
- Dynamics 365 Sales and Customer Service licenses with Copilot entitlements.
- System Administrator role in the Dynamics 365 environment.
- Microsoft 365 Copilot or Copilot for Sales/Service add-on license.

## Roles and permissions
- Lab environment: System Administrator role.
- Real-world deployments: System Administrator to enable Copilot features; end users require appropriate licenses.

## Estimated time

Duration: **20–30 minutes**
