## Task 03: Access the Demo Hub website and select a workshop

Now that you have a valid demo tenant, you can add the demo environment that your workshop requires. After you select a workshop, subsequent tasks will present steps to help you deploy the correct demo and configure prerequisites.

In this task, you select the workshop for which you plan to register. Customized instructions to prepare for the selected workshop will display.

---

### Which learning path do you plan to register for?

Choose your path below and follow the corresponding exercises.

{: .highlight }
> **Option 1 — Sales Transformation with AI**
>
> Proceed to [Exercise 02: Create a Sales Transformation with AI Environment](../Ex02_Sales_Environment/Ex02_Sales_Environment.md) to provision a demo environment and configure prerequisites for the Sales learning path.
