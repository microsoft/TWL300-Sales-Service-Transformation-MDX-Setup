## Task 06: Create an app registration and application user

The app user is a proxy user that performs actions on behalf of the Sales Qualification agent. The agent will read and respond to communications using this user. Without configuring an app user and performing these steps, some features of the Sales Qualification agent will not work. 

**Estimated time to complete this task**: 

- Hands-on: 3-5 minutes

### 01: Create the app registration

-  Open a web browser window and go to `https://portal.azure.com`.

-  Sign in by using the demo tenant administrative credentials:

Admin name: `admin@<YourTenantName>.onmicrosoft.com` 

-  Search for `App registrations` and then select the **App registrations** service.

![y8s5jvu1.jpg](../../media/y8s5jvu1.jpg)

-  Select **+ New registration**.

![9a1f3o08.jpg](../../media/9a1f3o08.jpg)

-  In the **Name** field, enter `Case agent`.

-  In the **Supported account types** section, select **Accounts in this organizational directory only (Single tenant)**.

![eopq22rp.jpg](../../media/eopq22rp.jpg)

-  Select **Register**.

-  Wait for registration to complete. Then, in the left pane for the registration, expand the **Manage** section and select **Certificates and secrets**.

![mcvh842w.jpg](../../media/mcvh842w.jpg)

-  Select **New client secret**.

![9f1acz4d.jpg](../../media/9f1acz4d.jpg)

-  In the **Description** field, enter `Used for the case management agent` and then select **Add**.

![yt5qg29k.jpg](../../media/yt5qg29k.jpg)

![bo9pq3z2.jpg](../../media/bo9pq3z2.jpg)

### 02: Create an application user

-  Return to the **Power Platform admin center** browser window.

-  In the **Manage** pane, select **Environments**.

![yri3zkh8.jpg](../../media/yri3zkh8.jpg)

-  On the **Environments** page, select your demo environment.

-  On the command bar at the top of the page, select **Settings**.

![cxcrvwx6.jpg](../../media/cxcrvwx6.jpg)

-  Expand **Users + permissions** and then select **Application users**.

![fo15ats1.jpg](../../media/fo15ats1.jpg)

-  Select **+ New app user**.

![958psir7.jpg](../../media/958psir7.jpg)

-  On the **Create a new app user** pane, in the **App** section, select **Add an app**.

![bq15nbgu.jpg](../../media/bq15nbgu.jpg)

-  Search for and select the `case agent` application user and then select **Add**.

![v6cbek4a.jpg](../../media/v6cbek4a.jpg)

![nzj71x9r.jpg](../../media/nzj71x9r.jpg)

-  In the **Business unit** field, enter `org` to search for business units and then select the existing business unit.

-  In the **Security roles** field, select the pencil icon (**Edit**).

![9vthzwea.jpg](../../media/9vthzwea.jpg)

-  Select **Customer Service Representative** and then select **Save**.

-  In the **Role assignment confirmation** dialog, select **Save**.

![w5brfdm3.jpg](../../media/w5brfdm3.jpg)

-  In the **Create a new app user** pane, select **Create**.

![m9z1rugv.jpg](../../media/m9z1rugv.jpg)

![0gyry8k0.jpg](../../media/0gyry8k0.jpg)
