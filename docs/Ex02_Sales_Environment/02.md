## Task 02: Add required users to the environment

Before you create agents, you need to add users to your organization. In this task, you'll add users to the demo environment.

**Estimated time to complete this task**: 

- Hands-on: 15-20 minutes

> 
>   As part of provisioning the demo environment, the system may have added some users for you. As you work through this task, if a user already exists, check to see that it's set up according to these steps. Add new users as needed.

> 

-  Open a web browser and go to `aka.ms/ppac`.

-  Sign in by using your demo admin credentials for the tenant that you created in Exercise 01.

![53nxl138.jpg](../../media/53nxl138.jpg)

-  In the left pane, select **Manage**.

![czrsuw6o.jpg](../../media/czrsuw6o.jpg)

-  In the **Manage** pane, select **Environments**.

![yri3zkh8.jpg](../../media/yri3zkh8.jpg)

-  On the **Environments** page, select your demo environment.

-  On the details page for the environment, in the **Access** section, under **Users**, select **See all**.

![y1cydmyt.jpg](../../media/y1cydmyt.jpg)

-  Compare the list of users in the table below with the list of users that are associated with the environment.

-  Leave Power Platform admin center open. You'll use the admin center again in the next task.
