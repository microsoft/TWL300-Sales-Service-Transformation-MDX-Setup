## Task 03: Add Copilot capacity credits

Sales agents use AI tokens to assist agents. In this task, you'll ensure that you dedicate Copilot Studio credits to your environment.

**Estimated time to complete this task**: 

- Hands-on: 3-5 minutes

-  Open a web browser and go to `aka.ms/ppac`.

-  Sign in by using your demo admin credentials for the tenant that you created in Exercise 01.

![53nxl138.jpg](../../media/img_008.jpg)

-  In the left pane, select **Licensing**.

![phj8x10o.jpg](../../media/img_017.jpg)

-  On the **Licensing** page, select **Copilot Studio**.

![o0t7q7kf.jpg](../../media/img_018.jpg)

-  In the **Copilot Studio** section, select **Manage Copilot Credits**.

![r1vnohne.jpg](../../media/img_019.jpg)

-  In the **Manage capacity** pane, select your demo envionrment.

-  In the **Copilot Credits** field, enter the maximum available credits.

-  In the **Capacity overages** section, select **Draw from the available capacity in my tenant**.

![3b0wz21a.jpg](../../media/img_020.jpg)

![ez1td9p1.jpg](../../media/img_021.jpg)

-  Select **Save**.
