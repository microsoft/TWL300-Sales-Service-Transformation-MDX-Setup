# Configure environment settings

In this section, you'll enable AI prompts and configure the modern look for your demo environment.

---

## Task 04: Enable AI prompts

1. Open a web browser and go to `aka.ms/ppac`.

2. Sign in by using your demo admin credentials for the tenant that you created in Exercise 01.

3. In the **Manage** pane, select **Environments**.

    ![yri3zkh8.jpg](../../media/yri3zkh8.jpg)

4. On the **Environments** page, select your demo environment.

5. On the command bar at the top of the page, select **Settings**.

    ![cxcrvwx6.jpg](../../media/cxcrvwx6.jpg)

6. Expand **Products** and then select **Features**.

    ![rjvy7sig.jpg](../../media/rjvy7sig.jpg)

7. Move down the page and locate the **AI prompts** section. 

8. Set the value of the **Enable the AI prompts feature in Power Platform and Copilot Studio** feature to **On**.

    ![nlwwgisj.jpg](../../media/nlwwgisj.jpg)

    > 
    >   For recent deployments, the value for this setting may be **On** by default.

    > 

9. Select **Save**.

---

## Task 05: Enable the modern look

For many of the new features to display correctly, you'll need to make sure that you've configured the system to use the new look.

1. Open a web browser and go to `make.powerapps.com`.

2. Sign in by using the demo tenant administrative credentials:

    Admin name: `admin@<YourTenantName>.onmicrosoft.com` 

3. On the command bar, select **Environment** and then select your demo environment.

4. In the left pane, select **Apps**.

    ![rmr0lgbs.jpg](../../media/rmr0lgbs.jpg)

5. In the list of apps, select **Sales Hub**.

    ![xprnh5h9.jpg](../../media/xprnh5h9.jpg)

6. Select **Settings** and then select **Features**.

    ![ssu26q2o.jpg](../../media/ssu26q2o.jpg)

7. Enable **New look for model-driven apps** and then select **Save**.

    ![pld0jkhx.jpg](../../media/pld0jkhx.jpg)

8. Publish the app.

    > 
    >   The **Publish** button appears at the top right on the command bar and looks like a window with an upward facing arrow.

    > 

    ![oosp8k1c.jpg](../../media/oosp8k1c.jpg)
