## Task 04: Enable AI prompts

**Estimated time to complete this task**: 

- Hands-on: 3-5 minutes

1. Open a web browser and go to `aka.ms/ppac`.

2. Sign in by using your demo admin credentials for the tenant that you created in Exercise 01.

3. In the **Manage** pane, select **Environments**.

    ![yri3zkh8.jpg](../../media/yri3zkh8.jpg)

4. On the **Environments** page, select your demo environment.

5. On the command bar at the top of the page, select **Settings**.

    ![cxcrvwx6.jpg](../../media/cxcrvwx6.jpg)

6. Expand **Products** and then select **Features**.

    ![rjvy7sig.jpg](../../media/rjvy7sig.jpg)

7. Move down the page and locate the **AI prompts** section. 

8. Set the value of the **Enable the AI prompts feature in Power Platform and Copilot Studio** feature to **On**.

    ![nlwwgisj.jpg](../../media/nlwwgisj.jpg)

    > 
    >   For recent deployments, the value for this setting may be **On** by default.

    > 

9. Select **Save**.
