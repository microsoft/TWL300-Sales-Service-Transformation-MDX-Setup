### Task 6: Create a long-term forecast

Now that you have created a short-term forecast, you will create a long-term forecast.

-  In **Copilot Service Workspace**, select **Workforce Management** and then select **Forecasting**.

![fqusbcxf.jpg](../../media/img_154.jpg)

-  On the command bar, select **+ New** and then select **Long Term forecast scenario**.

![yv5x2zus.jpg](../../media/img_161.jpg)

-  Configure the forecast as follows:

**Name:** `Sept Long Term - Conversation`

- **Duration:** 180

- **Interval:** Long Term

![image021.png](../../media/img_162.png)

-  In the **Historical data** section, configure as follows:

Data source: External

- External data: Contoso Daily

![image022.png](../../media/img_163.png)

-  In the Forecast entity field, select **Conversation**.

-  On the command bar, select **Save**.

-  On the command bar, select **Run Forecast scenario**. This schedules the scenario to be run.

![u38659ti.jpg](../../media/img_159.jpg)

- Close the **Sept Long Term - Conversation** tab.

![image020.png](../../media/img_160.png)
