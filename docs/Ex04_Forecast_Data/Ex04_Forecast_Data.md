# Exercise 04: Configure forecast external data

In this exercise, you'll enable Workforce Management, create bookable resources and shift activity types, import forecast data, and configure short-term and long-term forecast scenarios.

## Estimated time

- **Hands-on:** 30–45 minutes
- **Workforce Management installation:** up to 1 hour

## Tasks

1. Enable Workforce Management
2. Create bookable resources
3. Create shift activity types
4. Import daily and intraday external data
5. Create short-term forecasts
6. Create a long-term forecast
7. Configure short-term case forecasts
8. Configure long-term case forecasts
