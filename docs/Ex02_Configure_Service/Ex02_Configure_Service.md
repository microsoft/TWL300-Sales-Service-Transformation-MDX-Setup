# Exercise 02: Configure Dynamics 365 Customer Service

## Exercise learning objectives
- Configure the foundational settings of Dynamics 365 Customer Service.
- Set up queues, routing rules, and case management.
- Configure Service Level Agreements (SLAs) and entitlements.
- Enable omnichannel communication channels.

## Licensing and environment
- Dynamics 365 Customer Service license (Professional or Enterprise).
- System Administrator or Customer Service Manager role in the Dynamics 365 environment.

## Roles and permissions
- Lab environment: System Administrator role.
- Real-world deployments: Customer Service Manager or System Administrator role to configure entities and routing rules.

## Estimated time

Duration: **30–40 minutes**
